# BankApp

**Database**
- Relational Database (MySQL)

**Backend**
- Java
- Spring framework 
  - Spring Data JPA with Hibernate implementation
  - Dependency injection (Inversion of Control)
  - Data Transfer Object
  - Spring Security
  - Authentication and authorization
  
**Frontend**
- Javascript
- React
  - ES6
  - SPA
  - JSX
  - Sorting, Pagination
  - Bootstrap
